# dot-files

A repository containing configuration files and guides for Mota framework development with Cursor AI.

## Overview

This repository provides:

- Cursor AI rules for Mota framework development
- Claude AI guide for generating Mota workflows
- Configuration files for optimal development experience

## Contents

- `.cursor/rules/motia-rules.mdc`: Cursor AI rules for working with Mota framework components
- `CLAUDE.md`: Comprehensive guide for Claude AI to generate Mota workflows

## Getting Started

1. Clone this repository
```bash
git clone https://github.com/motia/dot-files.git
```

2. Copy the configuration files to your Mota project
```bash
cp -r .cursor your-motia-project/
```

3. Use Cursor AI with the provided rules to enhance your Mota development experience

## Features

- **Mota Rules**: Guidelines for Cursor AI to understand Mota framework concepts including:
  - Core components (steps, flows, events)
  - Step types (API, Event, Cron, NOOP)
  - UI components and styling
  - Best practices and testing workflows

- **Claude AI Guide**: Comprehensive documentation on how to use Claude for generating Mota workflows with:
  - Step templates for TypeScript, JavaScript, Python, and Ruby
  - Configuration patterns
  - State management conventions
  - Example prompts and implementations

## Usage

When using Cursor AI with these configurations, you'll get enhanced assistance for:

- Creating properly structured steps
- Following Mota naming conventions
- Implementing correct event patterns
- Building flows that adhere to best practices
- Generating consistent UI components
