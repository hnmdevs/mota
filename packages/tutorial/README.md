# @imoogle/tutorial

Mota tutorial package, provides a step-by-step guide to learn how to use Mota's ecosystem through Workbench.

## Setup

If you are starting a new project, simply run the create command and the tutorial will be included by default:

```bash
npx mota@latest create
```

For existing Mota project, using the latest version of the Mota CLI, download the tutorial using the following command:

```bash
npx mota@latest generate tutorial-flow
```
